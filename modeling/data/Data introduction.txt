1. This data comes from MAFA dataset and WIDER FACE dataset.
2. There are 6120 pictures in the training set, 3006 pictures from MAFA (basically all pictures with masks),
    There are 3114 pictures of WIDER Face (basically all pictures without a mask).
3. There are a total of 1839 images in the verification set, including 1059 images from MAFA and 780 images from WIDER Face
4. MAFA data set link: http://www.escience.cn/people/geshiming/mafa.html
5. WIDER Face data set link: http://shuoyang1213.me/WIDERFACE/
